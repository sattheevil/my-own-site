Проєкт на p5.js: Печиво клікер

Огляд проєкту

Цей проєкт демонструє гру про клікання печиво, яким ви можете заробляти печиво та зробити оновлення печива для більше печиво.

Мета проєкту:

Створити інтерактивну платформу для розваги.

Основні завдання:

  • Заробляти як найбільше печиво
  • Пройти повністю гру
  • Накопити як найбільше поінтів престижів

Технічна реалізація

Використані технології:

  • p5.js для графіки та анімації
  • JavaScript для логіки взаємодії
  • HTML5/CSS3 для структури та стилізації

Інноваційні аспекти:

  • Інтерактивні елементи, що реагують на дії користувача
  • Динамічні зміни кольорів та форм

Особливості проєкту

  • Інтерактивне печиво
  • Автоматична генерація печиво
  • Престижі, яке все заново починається з бонусами

Керування

  • Клік миші - отримання печива, прокачування печива, зроблення престижа
